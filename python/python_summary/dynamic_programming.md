<br/>

> 탑다운 (피보나치)
``` python
# 계산 결과를 담을 리스트
d = [0] * 100

def fibo(x):
    # 종료조건
    if x == 1 or x == 2:
        return 1

    # 이미 수행한 계산에 대해선 계산 결과 반환
    if d[x] != 0:
        return d[x]

    # 수행하지 않은 계산이면 계산값 반환
    d[x] = fibo[x-1] + fibo[x-2]
    return d[x]
```

> 바텀업 (피보나치)
```python
# 계산 결과를 담을 리스트
d = [0] * 100

d[1] = 1
d[2] = 2
n = 99

# 반복문으로 피보나치 구현
for i in range(3, n+1):
    d[i] = d[i-1] + d[i-2]

print(d[n])
```
